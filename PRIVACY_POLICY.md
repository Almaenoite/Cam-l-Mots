# Règles de Confidentialité - CaméléMots

Dernière mise à jour : 26 avril 2026

L'extension **CaméléMots** est conçue pour respecter la vie privée de ses utilisateurs. 

## 1. Collecte des données
L'extension **CaméléMots ne collecte, ne stocke et ne transmet aucune donnée personnelle**. 
- Aucune information sur votre navigation n'est enregistrée.
- Aucun contenu des pages que vous visitez n'est envoyé à des serveurs externes.
- Toutes les transformations de texte (coloration des syllabes) sont effectuées localement sur votre ordinateur, à l'intérieur de votre navigateur.

## 2. Utilisation des permissions
L'extension utilise les permissions suivantes uniquement pour son fonctionnement pédagogique :
- **Storage** : Pour mémoriser vos préférences de réglages (couleurs choisies, mode de lecture). Ces données restent sur votre ordinateur.
- **ActiveTab & Scripting** : Uniquement pour appliquer la coloration sur la page que vous consultez, et seulement lorsque vous activez l'extension via son bouton.

## 3. Communication
Puisque nous ne collectons aucune donnée, nous n'avons aucun moyen de vous contacter, de vendre vos informations ou de vous suivre sur internet.

## 4. Contact
Si vous avez des questions concernant ces règles, vous pouvez consulter le code source de l'extension qui est transparent et ouvert.
